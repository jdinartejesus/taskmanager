# taskmanager
Small app to manage time and tasks using pure javascript (ES6) with purpose of learning the language.

## Build

```
$ git clone git@github.com:jdinartejesus/taskmanager.git
$ cd taskmanager
$ npm install
$ npm run build
```

## Development

```
$ git clone git@github.com:jdinartejesus/taskmanager.git
$ cd taskmanager
$ npm install
$ npm run develop
```
